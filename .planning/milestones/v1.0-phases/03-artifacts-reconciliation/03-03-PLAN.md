    Create diligent/diligent/helpers/reconcile_anchors.py as a PURE function module. No imports from click, pathlib, os, or any I/O library. Only imports: dataclasses, datetime.date, and diligent.state.models types.

    Define dataclasses:
    - `StaleFactInfo`: key (str), old_value (str), new_value (str), source_id (str), days_stale (int), category (str: "value_changed" | "source_superseded" | "flagged"), fact_date (str), superseding_source_id (Optional[str], for source_superseded category)
    - `StaleArtifact`: path (str), workstream (str), value_changed (list[StaleFactInfo]), source_superseded (list[StaleFactInfo]), flagged (list[StaleFactInfo]), is_stale property (bool), is_advisory property (bool)

    Implement `compute_staleness`:
    ```python
    def compute_staleness(
        artifacts: list[ArtifactEntry],
        facts: dict[str, FactEntry],
        sources: list[SourceEntry],
        workstream: Optional[str] = None,
    ) -> list[StaleArtifact]:
    ```

    Algorithm:
    1. Build superseded_by index: `dict[str, SourceEntry]` mapping old_source_id to the SourceEntry that superseded it (from sources[].supersedes field -- this is reverse lookup)
    2. If workstream filter provided, filter artifacts to matching workstream
    3. For each artifact:
       a. For each referenced truth key:
          - Skip if key not in facts dict
          - Get the FactEntry
          - **Value changed**: compare fact.date > artifact.last_refreshed (ISO string comparison works for YYYY-MM-DD). If true, add StaleFactInfo with category="value_changed", old_value from supersedes[0].value if available (else ""), new_value=fact.value, source_id=fact.source, days_stale=(today - fact_date).days
          - **Source superseded**: check if fact.source is in superseded_by index. If yes, check superseding source's date_received > artifact.last_refreshed. If true, add StaleFactInfo with category="source_superseded", superseding_source_id
          - **Flagged**: if fact.flagged is not None, add StaleFactInfo with category="flagged"
       b. Sort value_changed and source_superseded by fact_date descending (most recent first)
    4. Return list of StaleArtifact (include all artifacts if no filter, or only those with non-empty lists if filtering is caller's concern -- return all and let CLI filter)

    Key constraints:
    - Source-superseded ONLY fires if superseding source date > artifact.last_refreshed. This prevents false positives from historical supersedes events that happened before the artifact was refreshed.
    - A fact can appear in BOTH value_changed and source_superseded if both conditions are true.
    - Flagged facts NEVER affect is_stale. They are advisory only.
    - days_stale for value_changed/source_superseded: (date.today() - date.fromisoformat(fact_date)).days
    - days_stale for flagged: (date.today() - date.fromisoformat(flagged_date)).days
  </action>
