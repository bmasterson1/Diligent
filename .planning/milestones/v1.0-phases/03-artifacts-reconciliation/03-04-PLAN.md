    ```python
    import re

    CITATION_PATTERN = re.compile(r"\{\{truth:([a-zA-Z0-9_]+)\}\}")

    def scan_docx_citations(path: str) -> list[str]:
        """Extract truth key references from a .docx file.

        Scans paragraph text for {{truth:key_name}} citation tags.
        Lazy-imports python-docx to keep CLI startup fast.

        Returns sorted unique list of referenced truth keys.
        Returns empty list if file cannot be read.
        """
        try:
            from docx import Document
            doc = Document(path)
            keys: set[str] = set()
            for para in doc.paragraphs:
                for match in CITATION_PATTERN.finditer(para.text):
                    keys.add(match.group(1))
            return sorted(keys)
        except Exception:
            return []
    ```

    Update artifact register in artifact_cmd.py to integrate scanner:
    - After path normalization, check if path ends with `.docx` (case-insensitive)
    - If .docx:
      - Import scan_docx_citations (lazy, inside the register function body)
      - Resolve the absolute path of the docx file (deal_root / normalized_path)
      - Run scanner: `found_keys = scan_docx_citations(str(abs_docx_path))`
      - If --references provided:
        - Store --references in artifact.references (authoritative)
        - Store scanner findings in artifact.scanner_findings
        - If scanner found keys not in --references, print advisory: "Scanner also found these keys you didn't list: {extra_keys}"
      - If --references NOT provided:
        - If scanner found keys: print "Scanner found these truth keys: {found_keys}. Re-run with --references to confirm." Exit 1
        - If scanner found nothing: print "No citation tags found in .docx. Provide --references." Exit 1
    - If not .docx: scanner does not run. --references is the only path.

    IMPORTANT: --references is required for non-.docx files. For .docx files, make --references optional (the scanner provides a fallback path, but still exits 1 requiring analyst to explicitly provide --references). This means the Click option `--references` should NOT be `required=True` globally -- instead validate it in the command logic: if not .docx and no references, error.
