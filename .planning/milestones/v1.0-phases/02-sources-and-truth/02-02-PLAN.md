    - Test: ingest creates a source entry in SOURCES.md with all metadata fields populated
    - Test: first ingest in empty SOURCES.md produces {DEAL_CODE}-001
    - Test: subsequent ingests produce monotonically increasing IDs ({DEAL_CODE}-002, 003, etc.)
    - Test: source ID derives from SOURCES.md max, not a counter file (self-healing after manual edits)
    - Test: path stored as relative to deal folder root (parent of .diligence/), not absolute
    - Test: --date defaults to today when omitted
    - Test: --parties accepts comma-separated string, stored as list
    - Test: --workstream accepts a single workstream tag
    - Test: --supersedes accepts a prior source ID string
    - Test: --notes accepts free text
    - Test: --json outputs structured result with source_id, path, metadata
    - Test: ingest fails with error if the referenced file does not exist
    - Test: ingest fails with error if .diligence/ directory not found
  </behavior>
  <action>
    Create sources_cmd.py with a Click group and ingest subcommand.

    1. `sources_cmd` is a `@click.group("sources")` with docstring "Manage source documents."

    2. `_next_source_id(sources: SourcesFile, deal_code: str) -> str` helper: scan all source IDs matching `{deal_code}-` prefix, parse the numeric suffix, return `{deal_code}-{max+1:03d}`. If no existing sources, return `{deal_code}-001`.

    3. `_resolve_relative_path(file_path: Path, deal_root: Path) -> str` helper: resolve file_path to absolute, then compute relative path from deal_root. Deal root is `.diligence/..` (parent of the .diligence directory). Return as posix string for cross-platform consistency in SOURCES.md.

    4. Ingest subcommand:
       - Arguments: PATH (required, type=click.Path(exists=True))
       - Options: --date (default: today ISO), --parties (comma-separated string), --workstream (string), --supersedes (string), --notes (string), --json (flag)
       - Flow: find .diligence/ in cwd, read config for deal_code, read SOURCES.md, generate next ID, resolve relative path, create SourceEntry, append to sources list, write_sources back, output result
       - Validate: if --supersedes is provided, verify that source ID exists in SOURCES.md (warn if not, do not block)
       - Error: if .diligence/ not found, print error and exit 1. If file at PATH does not exist, Click handles via exists=True.

    5. Parse --parties: split on comma, strip whitespace, filter empty strings. Store as list.
  </action>

    <automated>cd diligent && python -m pytest tests/test_ingest.py -x -q</automated>
  </verify>
  <done>
    `diligent ingest <path>` registers a source document with metadata flags. Source IDs are monotonic, zero-padded, derived from SOURCES.md. Paths stored as relative. --json output works.
  </done>
