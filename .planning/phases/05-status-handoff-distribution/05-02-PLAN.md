        """Copy text to system clipboard. Returns True on success, False on failure. Never raises."""

    - Windows: subprocess.run(["clip.exe"], input=text, text=True, timeout=5, capture_output=True)

    **Instruction Header:**
    Static template string with `${DEAL_CODE}` substitution using string.Template. Content per CONTEXT.md locked decision:
    - Title: `# {DEAL_CODE} Deal Context`
    - Brief explanation of diligent and its concepts (TRUTH.md, SOURCES.md, ARTIFACTS.md, QUESTIONS.md, WORKSTREAMS.md)
    - Editorial principles: source every claim, never invent values, flag discrepancies, no AI template phrasing
    - `---` separator after header

    **Time Window Logic:**
    - Default window: 14 days (double the config.recent_window_days default of 7). Read config.recent_window_days and double it for the default.
    - `--since` overrides the default: parse with `parse_since(since_str, default_days)` from time_utils.py
    - `--everything` sets cutoff to a very early date (date(2000, 1, 1)) to include everything

    **Content Sections (markdown output):**

    1. **Instruction header** (always first, from template)
    2. **Deal context**: Read DEAL.md, render as markdown section `## Deal`. Include all frontmatter fields formatted as readable text.
    3. **Workstreams**: Read WORKSTREAMS.md, render as `## Workstreams`. Full content, one subsection per workstream with status.
    4. **Recent Truth Facts**: Read TRUTH.md, filter by `is_recent(fact.date, cutoff)` OR `fact.flagged is not None` (flagged always included). Render as `## Truth (Recent)`. For each fact: key, value, source, date. If --everything, section title is `## Truth (All)`.
    5. **Recent Sources**: Read SOURCES.md, filter by `is_recent(src.date_received, cutoff)`. Render as `## Sources (Recent)`. For each source: id, path, date_received, parties.
    6. **Open Questions**: Read QUESTIONS.md, filter status == "open" (always included regardless of date). Render as `## Open Questions`. For each: id, question, workstream, owner.
    7. **Stale Artifacts**: Use compute_staleness (lazy import) to find stale artifacts (always included regardless of date). Render as `## Stale Artifacts`. For each: path, changed facts, days stale.
    8. **Task Summaries**: For each workstream, scan .diligence/workstreams/{ws}/tasks/ for task directories. Find the most recent SUMMARY.md (by directory name or modified time). If it has content, include under `## Recent Task Summaries`. One per active workstream.

    **JSON mode:** Build a dict with sections as keys, content as values. Output with json.dumps.

    **Clipboard (--clip):**
    After building the full output string, always print to stdout. If --clip, also call copy_to_clipboard(output). Print "Copied to clipboard." on True, "Clipboard not available on this system" on False (to stderr via click.echo with err=True).
