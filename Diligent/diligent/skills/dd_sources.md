# dd:sources

Register and manage source documents in SOURCES.md.

## When to use

- A new document has been received (CIM, financials, contracts, etc.)
- Checking what sources exist and their metadata
- Comparing two versions of a document to see what changed
- Updating source metadata or tracking document lineage

## Commands

### ingest
```bash
{{DILIGENT_PATH}} ingest <path> [--supersedes <source-id>] [--workstream <ws>] [--parties <text>] [--notes <text>] [--json]
```
Register a new source document. Copies or links the file and creates a SOURCES.md entry.
Source IDs are auto-generated (DEAL_CODE-NNN). If `--supersedes` is specified, the new
source replaces the referenced one in the supersedes chain. For Excel files, auto-diff
runs against the superseded document.

### sources list
```bash
{{DILIGENT_PATH}} sources list [--json]
```
List all registered sources with ID, filename, date, and workstream.

### sources show
```bash
{{DILIGENT_PATH}} sources show <source-id> [--json]
```
Display full metadata for a source: path, parties, notes, supersedes chain,
and ingestion date.

### sources diff
```bash
{{DILIGENT_PATH}} sources diff <source-id-a> <source-id-b> [--verbose] [--json]
```
Compare two source documents. Shows sheet-level and cell-level differences
for Excel files. Use `--verbose` for full cell-by-cell output.

## Rules

- Source IDs are auto-generated by scanning SOURCES.md for the current max. No counter file.
- The supersedes chain tracks document lineage. Insert at position 0 (most recent first).
- Relative paths are stored as POSIX strings for cross-platform OneDrive sync.
- Source documents are read-only after ingestion. Do not modify ingested files.
- Auto-diff only fires for Excel-to-Excel comparisons.

## Common workflows

### Ingest a new source
```bash
{{DILIGENT_PATH}} ingest "./docs/CIM v2.pdf" --workstream financial --notes "Updated CIM from broker"
```

### Compare two versions of a document
```bash
{{DILIGENT_PATH}} sources diff ARR-001 ARR-003 --verbose
```

### Check source history
```bash
{{DILIGENT_PATH}} sources show ARR-001
```
